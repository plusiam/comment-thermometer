# 💬 댓글 온도계

## 프로젝트 소개

초등학교 4학년 도덕 수업 "디지털 세상에서 함께 살아가기" 프로젝트의 웹 학습지입니다.

학생들이 댓글이 주는 영향을 인식하고, 말이 타인에게 미치는 영향을 시각적으로 이해할 수 있도록 설계된 인터랙티브 웹 도구입니다.

## 📚 교육 목표

1. **개념 이해**: 말이 주는 다양한 영향 이해 (상처, 응원, 무관심)
2. **비판적 사고**: 다양한 댓글을 분석하고 분류하는 경험을 통한 판단력 향상
3. **메타인지**: 자신의 언어 습관을 성찰하고 책임 있는 언어 사용의 필요성 인식
4. **디지털 시민성**: 온라인 공간에서의 올바른 소통 태도 함양

## 🎯 주요 기능

### 1. 스토리 기반 도입
- 슈퍼토끼 재빨라의 뉴스 이야기로 흥미 유발
- 학생들이 공감할 수 있는 상황 설정

### 2. 드래그 앤 드롭 분류 활동
- 12개의 다양한 댓글 제시
- **💔 상처주는 말** - 기분 나쁘게 하는 말
- **💚 응원하는 말** - 힘이 되고 기분 좋게 하는 말
- **😐 무관심한 말** - 별 생각 없이 하는 말
- 직관적인 인터페이스로 쉬운 조작

### 3. 유연한 채점 시스템 ⭐
- **상처주는 말**과 **무관심한 말**은 섞어도 정답 처리
- **응원하는 말**만 정확히 구분하면 됨
- 긍정적 말과 부정적 말의 구분이 핵심!

### 4. 학습 결과 이미지 저장 📸 **NEW!**
저장되는 내용:
- ✅ 분류 결과 차트 (💔💚😐 개수)
- ✅ 이해도 점수 (%)
- ✅ 피드백 메시지
- ✅ **성찰 질문 답변** (작성한 내용만)

파일명: `댓글온도계_결과_YYYYMMDD_HHMM.png`

### 5. 성찰 질문
- 3가지 핵심 질문을 통한 깊이 있는 성찰
- 자유롭게 생각을 기록할 수 있는 공간
- **작성한 답변은 이미지에 함께 저장됨!**

## 💻 사용 방법

2. 또는 GitHub Pages로 배포하여 링크 공유

```bash

git clone [repository-url]

# 폴더로 이동

# index.html 파일을 브라우저에서 열기
```

## 📖 활동 순서

1. **도입**: 슈퍼토끼 재빨라의 뉴스 읽기
2. **탐구**: 12개의 댓글을 세 가지 유형으로 분류하기
3. **확인**: 결과 보기 및 피드백 받기
4. **성찰**: 3가지 질문에 답하며 생각 정리하기 💭
5. **저장**: 📸 이미지로 저장하기 버튼 클릭 ⭐
6. **반복**: 다시 하기 버튼으로 재도전 가능

## 📸 이미지 저장 기능 상세

### 저장되는 내용 예시
```
🎯 나의 분류 결과
💔 상처주는 말: 3개
💚 응원하는 말: 5개  
😐 무관심한 말: 4개

🎉 정말 잘했어요!
나의 이해도: 95%
💡 응원하는 말과 부정적인 말을 잘 구분했어요.

💭 나의 생각

❓ 상처주는 말과 무관심한 말은 어떻게 다를까요?
상처주는 말은 일부러 나쁜 마음으로 하는 말이고,
무관심한 말은 별 생각 없이 했지만 듣는 사람은 
상처받을 수 있는 말인 것 같아요.

❓ 재빨라가 이런 댓글들을 본다면 어떤 기분일까요?
슬프고 화가 날 것 같아요. 특히 악의적인 댓글을 
보면 정말 기분이 나쁠 것 같아요.

❓ 댓글을 달기 전에 우리가 생각해야 할 것은 무엇일까요?
상대방이 이 말을 듣고 어떤 기분일지 먼저 생각하고,
나쁜 감정을 주는 말은 하지 말아야 할 것 같아요.
```

### 작동 방식
```
1. 성찰 질문에 답변 작성
   ↓
2. "결과 확인하기" 버튼 클릭
   ↓
3. "📸 이미지로 저장하기" 버튼 클릭
   ↓
4. 작성한 답변이 있으면 자동으로 포함됨
   (답변이 없는 질문은 제외)
   ↓
5. PNG 이미지로 저장
```

## 🎨 댓글 예시

### 💔 상처주는 말 (기분 나쁘게 하는 말)
- "잘난 척하더니 꼴좋다!"
- "너무 자만했잖아. 당연한 결과야."
- "재빨라는 이제 끝났어"

### 💚 응원하는 말 (힘이 되고 기분 좋게 하는 말)
- "재빨라도 수고했어!"
- "다음엔 더 잘할 수 있을 거야!"
- "실수할 수도 있지! 다음 기회에 힘내!"
- "재빨라 응원했는데 아쉽네요. 하지만 최선을 다한 모습이 멋있었어요!"

### 😐 무관심한 말 (별 생각 없이 하는 말)
- "어떻게 거북이한테 질 수가 있지?"
- "재빨라의 시대는 끝났다"
- "거북이가 이겼네 ㅋㅋㅋㅋ"
- "낮잠 자다니 프로정신이 없네"

## 🎓 교육적 활용

### 시나리오 1: 완전한 학습 기록
```
학생: 1. 댓글 분류
     2. 성찰 질문에 깊이 있게 답변 작성
     3. 이미지로 저장
     → 분류 결과 + 나의 생각이 모두 담긴 완전한 기록!
```

### 시나리오 2: 성장 과정 비교
```
1주차: 간단히 답변 작성 → 이미지 저장
       ↓
토론 및 학습
       ↓
2주차: 더 깊이 생각하며 답변 → 이미지 저장
       ↓
두 이미지 비교: 생각의 깊이가 어떻게 달라졌을까?
```

### 시나리오 3: 학부모 소통
```
자녀: 성찰 답변까지 작성 → 이미지 저장 → 부모님께 전송
부모님: "오늘 이런 걸 배웠구나! 네 생각이 참 깊어졌네"
       → 자녀의 도덕적 성장 확인
```

### 시나리오 4: 포트폴리오
```
학기말 포트폴리오에:
- 1차 저장 이미지 (초기 생각)
- 2차 저장 이미지 (발전된 생각)
- 3차 저장 이미지 (완성된 생각)
→ 자신의 성장 과정을 한눈에!
```
